# 蓝乐 Agent 控制台演示

这是蓝乐 AI 音乐创作平台的前端概念演示。

视觉方向调整为 Ins 高级风音乐工具界面：

- 浅色高级背景。
- 柔和粉橙、蓝紫渐变。
- 玻璃质感卡片。
- 更强留白和轻阴影。
- 登录入口。
- 一键洗歌教程。
- 左侧自定义二创。
- 中间榜单分类。
- 右侧热歌排行。

当前阶段定位：

```text
人操作多个 Agent
```

用户在控制台中分别操作：

- 酷狗爬虫 Agent
- 音乐分析 Agent
- Suno 创作 Agent

后续升级方向：

```text
总调度 Agent 调用多个专业 Agent
```

也就是用户只提出目标，总调度 Agent 自动安排爬虫、分析、创作等专业 Agent 协作完成任务。
