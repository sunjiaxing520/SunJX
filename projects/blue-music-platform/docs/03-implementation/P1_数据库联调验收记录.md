## 已知问题

- `docker-compose.yml` 中的 `version` 字段会被新版 Docker Compose 提示为 obsolete，当前不影响运行。
- Windows 更新或系统设置可能关闭 `VirtualMachinePlatform`，若 Docker Desktop 报 `Virtualization support not detected`，需要重新启用该功能并重启。
