# 后续增强设想

本文档记录蓝乐 AI 音乐创作平台的后续增强方向。

这些内容属于初步设想，不纳入当前首期交付范围。等基础项目跑通、客户确认价值后，再单独评估周期和费用。

## 一、当前阶段边界

首期先做固定、可控、容易验收的 Agent。

当前 Agent 可以理解为自动化任务机器人：

```text
爬虫 Agent：自动采集酷狗热歌榜数据
分析 Agent：按固定规则或固定提示词分析歌曲特征
作词 Agent：根据主题、风格和基础记忆生成歌词
创作 Agent：整理创作素材，提交给 Suno，跟踪状态，保存结果
```

其中创作 Agent 初期不等于自研音乐生成模型。

它的主要职责是：

```text
整理表单
提交任务
跟踪状态
回收结果
保存记录
```

真正的音乐生成能力暂时由 Suno 等外部工具提供。

## 二、高级 Agent 工作链

当前版本已经具备采集、分析、作词的线性工作链配置与自动传参。

后续可以升级为拖拽画布、条件分支、并行节点和自定义参数映射。

示例：

```text
酷狗榜单采集 Agent
-> 爆款趋势分析 Agent
-> 作词 Agent
-> 情绪氛围分析 Agent
-> Suno 创作 Agent
```

也可以配置成：

```text
酷狗榜单采集 Agent
-> 歌词主题分析 Agent
-> 短视频创作 Agent
```

这个功能的价值是：

- 客户可以自己组合流程。
- 不同 Agent 可以有不同擅长方向。
- 平台从固定流程升级为可配置创作工作台。

## 三、带记忆的增强 Agent

后续增强 Agent 可以拥有自己的记忆。

记忆可以包括：

```text
擅长风格
历史分析偏好
客户常用创作方向
客户常用歌词主题
客户喜欢的歌词表达方式
常用提示词模板
过往任务结果
用户反馈
```

举例：

```text
爆款分析 Agent：更关注传播性、副歌抓耳、短视频适配。
专业音乐分析 Agent：更关注结构、节奏、调性、编曲。
治愈系创作 Agent：更关注情绪、歌词氛围、人声质感。
作词 Agent：更关注歌词主题、语气、押韵、段落结构和客户偏好。
短视频创作 Agent：更关注开头 15 秒、节奏冲击、记忆点。
```

## 四、总调度 Agent

更高级的方向是增加一个总调度 Agent。

用户不再手动选择每一步，而是只说目标：

```text
参考最近热歌，做一首适合短视频传播的治愈系女声流行歌。
```

总调度 Agent 自动完成：

```text
理解目标
拆分任务
选择合适的专业 Agent
安排执行顺序
汇总结果
给用户反馈
```

示例流程：

```text
用户目标
-> 总调度 Agent
-> 酷狗爬虫 Agent
-> 爆款趋势分析 Agent
-> 作词 Agent
-> 情绪分析 Agent
-> Suno 创作 Agent
-> 结果回收与展示
```

## 五、对客户的讲法

可以这样讲：

```text
当前版本先提供可保存的线性流程，保证系统能跑通、能演示、能验收。
后续如果客户使用效果好，可以升级为带分支、并行和参数映射的高级工作链。
再往后，可以增加带记忆和 AI 推理能力的总调度 Agent，让用户只描述目标，系统自动安排多个 Agent 协作。
这些属于后续增强能力，可以单独评估周期和费用。
```

## 六、不纳入首期范围

以下内容暂不纳入首期：

- 拖拽式高级工作链、条件分支和并行节点。
- Agent 长期记忆系统。
- 总调度 Agent 自动拆任务。
- 多个增强 Agent 自动协作。
- 根据用户反馈自动优化 Agent 偏好。
- 多个作词 Agent 拥有不同记忆。

首期目标仍然是：

```text
酷狗榜单采集 -> 基础分析 -> AI 作词 -> Suno 创作任务 -> Dashboard 展示
```
