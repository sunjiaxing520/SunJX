# 蓝乐 AI 音乐创作平台 —— 报价方案精简版

## 1. 项目定位

蓝乐 AI 音乐创作平台是一套音乐创作辅助工作台。

首期核心链路：

```text
酷狗榜单采集 -> 音乐分析 -> AI 作词 -> Suno 创作 -> 结果展示
```

## 2. 基础版

基础版目标：跑通完整业务闭环，适合先演示、先验证、先交付。

建议报价：**6000 元起**。

包含功能：

| 模块 | 内容 |
|---|---|
| 用户系统 | 登录、基础权限 |
| 酷狗爬虫 Agent | 手动采集酷狗热歌榜 |
| 音乐分析 Agent | 分析风格、情绪、关键词 |
| 作词 Agent | AI 写歌词，支持基础偏好记忆 |
| Suno 创作 Agent | 整理素材，提交给 Suno，跟踪结果 |
| Dashboard | 展示榜单、分析、歌词、创作任务 |
| 前端 demo | 精致演示页面 |

交付周期：**4-5 周**。

## 3. 完整版

完整版目标：更适合长期使用和团队管理。

建议报价：**8000 元起**。

在基础版上增加：

| 增强模块 | 内容 |
|---|---|
| 多源爬取 | 酷狗之外扩展 QQ 音乐、网易云等 |
| 定时任务 | 自动定时采集榜单 |
| 可视化图表 | 风格占比、趋势、榜单变化 |
| 在线试听 | 平台内试听生成结果 |
| 创作版本管理 | 同一首歌保存多个版本 |
| 操作日志 | 记录用户操作，方便追踪 |
| 数据备份 | 数据库备份和恢复 |

交付周期：**6-8 周**。

## 4. 后续加费项目

以下不纳入基础版和完整版首期交付，后续单独评估报价。

| 项目 | 说明 |
|---|---|
| 自定义 Agent 工作链 | 客户自己选择多个 Agent 组成流程 |
| 多个作词 Agent | 不同作词 Agent 拥有不同风格和记忆 |
| 多个创作 Agent | 不同创作 Agent 适配不同创作方向 |
| 增强记忆系统 | 长期记录客户偏好、反馈和历史作品 |
| 总调度 Agent | 用户只说目标，系统自动调度多个 Agent |

## 5. 付款建议

建议采用分期付款：

| 节点 | 比例 | 说明 |
|---|---:|---|
| 启动款 | 30% | 确认需求和合同后支付 |
| 中期款 | 30% | 榜单采集、分析、作词链路可演示后支付 |
| 尾款 | 40% | 全部功能验收通过后支付 |

## 6. 重要边界

- 首期不自研音乐生成模型。
- 音乐生成能力暂时依赖 Suno。
- 平台负责数据采集、分析、作词、任务整理、提交、结果管理。
- 客户新增功能、改范围、增加平台数据源，需单独评估。
- Suno、服务器、域名、第三方接口费用由客户承担。
